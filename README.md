# Studies

This repository contains study materials and notes.

## Patanjali Yoga Sutras

This directory contains the complete Patanjali Yoga Sutras in Hindi with detailed explanations.

### Files

- `Patanjali Yoga Sutras_list.md` - Complete list of all 196 sutras organized by chapters (पाद)
- `Patanjali Yoga Sutras Hindi.md` - Detailed Hindi translations and explanations of all sutras

### Structure

The sutras are organized into four chapters (पाद):

1. **समाधिपादः** (Samadhi Pada) - 51 sutras
2. **साधनपादः** (Sadhana Pada) - 55 sutras  
3. **विभूतिपादः** (Vibhuti Pada) - 56 sutras
4. **कैवल्यपादः** (Kaivalya Pada) - 34 sutras

**Total: 196 sutras**

Each sutra includes:
- Sanskrit text
- Word-by-word meaning (शाब्दिक अर्थ)
- Complete meaning (संपूर्ण शाब्दिक अर्थ)
- Detailed explanation (भावार्थ)
